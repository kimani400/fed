WHAT IS A CSS.

CSS is the languge use to style an HTML Document and its stand for Cascading Styling Sheet

Difference Ways CSS can be inserted in to HTML

Below are some of the ways CSS can be inserted in to the;

    Inline Css; An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element.

    Internal CSS; An internal CSS is used to define a style for a single HTML page.

    External CSS An external style sheet is used to define the style for many HTML pages.
